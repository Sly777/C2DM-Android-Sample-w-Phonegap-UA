This is sample project for show how to get APID from javascript on Android App.


Used Libraries:

// Phonegap 1.1.0 : http://phonegap.com
// Sencha Touch 1.1.0 : http://www.sencha.com/products/touch
// Urban Airship : http://urbanairship.com

If see any error that says "not found library", Please add files from /libs folder to your project (not copy, include your project using Eclipse)

Dont forget to change key info on airshipconfig.properties file.
